# Port Scanner 

## Manual
```python3
./class_ports -h
```
### in the port_scanner you can write this differents ways to get ports: range, only one port, some ports

<br>

## range
```python3
./class_ports -l 1.1.1.1 -p 1-100
```
<br>

## one_port
```python3
./class_port -l 1.1.1.1 -p 80
```

<br>

## some_ports
```python3
./class_port -l 1.1.1.1 -p 20,80,443,22
```
